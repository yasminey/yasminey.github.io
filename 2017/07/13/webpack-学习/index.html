webpack 学习

Share

webpack

1.首先要安装 Node.js， Node.js 自带了软件包管理器 npm，Webpack 需要 Node.js v0.6 以上支持，建议使用最新版 Node.js。

用 npm 安装 Webpack：

$ npm install webpack -g
此时 Webpack 已经安装到了全局环境下，可以通过命令行 webpack -h 试试。

通常我们会将 Webpack 安装到项目的依赖中，这样就可以使用项目本地版本的 Webpack。

进入项目目录

确定已经有 package.json，没有就通过 npm init 创建

安装 webpack 依赖

$ npm install webpack –save-dev

2.在根目录下创建 webpack.config.js
配置webpack.config.js
module.exports= {

entry: ‘./index.js’,

output: {
path: __dirname+’/dist’,
filename: ‘bundle.js’
}
}
3.打包 webpack
4.配置开发环境:$ npm install webpack-dev-server -g
颜色 $ webpack-dev-server –progress –colors
监听: $ webpack-dev-server –progress –colors –watch(自动刷新,或者–inline)
它将在 localhost:8080 启动一个 express 静态资源 web 服务器，并且会以监听模式自动运行 webpack，在浏览器打开 http://localhost:8080/ 或 http://localhost:8080/webpack-dev-server/ 可以浏览项目中的页面和编译后的资源输出，并且通过一个 socket.io 服务实时监听它们的变化并自动刷新页面。
(1.先输入localhost:8080查看显示情况,在输入http://localhost:8080/webpack-dev-server后按提示点击到页面,有时候直接输入http://localhost:8080/webpack-dev-server/ 没有监听效果,需要按1操作)

5.index.js要识别style.css,要加载css-loader(引入style.css)和style-loader(识别写好的样式)
下载:npm install css-loader –save
识别style (css中的内容)
下载:npm install style-loader –save

功能:模块引入 实现监听 打包
