react 学习

发表于 2017-07-16

安装
create-react-app
$ npm install -g create-react-app
$ create-react-app my-app my-app是目录名字,自己起名字,不要和npm中的依赖重名
$ cd my-app/
$ npm start 启动项目
$ npm run bulid 打包
将自动生成的src中文件留下index.js,index.css,logosvg,其他删除

在index.js中首先引入两位带头大哥(这两个模块非常重要)

import React from ‘react’
import ReactDOM from ‘react-dom’

jsx==js+x ==JavaScript+xml

组件==对象

插值用{},插值里不能用if语句

在react中设置类名class,必须改成className (大写的是驼峰标识,第一个单词小写后面的单词大写:className)
react中的input中for属性,要写成htmlFor

组件:自定义组件首字母要大写,用于区分html标签
state ,要想修改state,要用setState({})
get方法只能读不可写,set可读可写
在react中 index.js都是入口文件,不做大量的操作
在使用 JavaScript classes 时，你必须调用 super(); 方法才能在继承父类的子类中正确获取到类型的 this 。
在组件中方法要确定this指的是当前的组件,绑定this用箭头函数,bind(this,’参数’)
父级往子集传值,在父级中自定义属性,子集用props
